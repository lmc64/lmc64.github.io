Women for Women Web Portal

C#, Javascript, HTML, CSS

I worked with a team of 6 to develop a web portal for the organization Women for Women. This organization connects women in war zones to sponsors who donate money to support them. Our application used computer vision to read hand-written text in the language of the women being supported and translate it into the language of their supporters, allowing them to more effectively communicate.

I worked primarily as a front-end developer on the project. I also did some of the work of a Program Manager, functioning as the main point of contact between WFW and my team and leading demos of the project.

Tunder Music Discovery Web Application

Javascript, HTML, CSS

Worked with a team of 4 engineers to develop a web application that provides a users with a new way to interact with music on Spotify The app provides them with previews of the newest tracks and allows them to like or dislike them. The app uses these likes and dislikes to store information about the user's music preferences and then creates playlists based on these preferences.

Simple Logo Program

Java, Java FX

I worked with a team of 3 others to design a turtle based coding environment that uses a simple version of the Logo programming language.

The program supports function creation, variable assignment, and loops with variable iteration patterns. It also allows the user to load preset turtle functions and also to save their own functions to file. It also allows the user to save and load their workspace so that the set of functions they have defined can be accessed again when they reload the program.

Cellular Automata Simulator

Java, Java FX, XML

I worked with a team of 2 others to design a program capable of simulating and displaying several different types of cellular automata.

Stats about the states of the cells in the simulation grid are displayed visually on a graph. The user can choose between a fire spreading simulation, a game of life simulation, a predator-prey simulation, a segregation simulation, and a slime mold simulation. The program gives the user the ability to set multiple variables about the situation being simulated, as the ability to change the simulation speed, to pause and then resume the simulation, and to move through the simulation step by step.

Sexual Assault Awareness Game

IN PROGRESS

Unity, C#

I worked with a team of 7 engineers and writers to develop a game to raise awareness about the problem of sexual assault on Duke's campus. The application allows the user to play from the point of view of a student at Duke and puts them in the position of a bystander in several difficult situations revolving around sexual harassment and assault. The game is designed to help users understand the unique points of view of their classmates and learn how to be a positive, active, and safe bystander.

I worked as both a full-stack developer on the game and a writer of the game's content.

Ebbinghaus Learning Retention App

IN PROGRESS

Javascript, AngularJS, HTML, CSS

I worked on a team of 4 engineers and writers to develop an app that would help students retain more of the information they learn. The app was specifically designed to combat the Ebbinghaus forgetting curve, which describes how information is lost over time when it is not used. The app allows teachers to send reminders to their students in the form of quizzes and gathers information from students about how well they retain the information that they learn.

I worked as a full-stack developer and program manager on the project.
